# CamDiv - Free Random Video Chat Platform

[![Website](https://img.shields.io/website?url=https%3A%2F%2Fcamdiv.com&label=camdiv.com)](https://camdiv.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**[CamDiv](https://camdiv.com)** is a modern, anonymous video chat platform designed to connect strangers worldwide instantly. It serves as a safe, registration-free alternative to Omegle, leveraging peer-to-peer WebRTC technology for secure, low-latency communication directly in the browser.
